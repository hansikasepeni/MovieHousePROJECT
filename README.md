
#  MovieHouse - Movie Search Web Application

MovieHouse is a responsive movie search web application built using **HTML, CSS, and JavaScript**. This project allows users to search and explore movies through a clean, modern, and user-friendly interface.

This project was developed as part of my frontend development journey to strengthen my understanding of web fundamentals, responsive design, and creating interactive user experiences using JavaScript.

## Features

- Search movies dynamically
- Display movie information with an interactive UI
- Responsive design for different screen sizes
- Clean and modern user interface
- Real-time user interactions using JavaScript
- Smooth and simple navigation

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- TMDB API

##  Learning Outcomes

Through this project, I gained practical experience in:

- Building responsive web pages from scratch
- Creating interactive user interfaces using JavaScript
- Working with APIs to fetch and display dynamic data
- Understanding DOM manipulation
- Improving frontend development and UI design skills

## Project Purpose

MovieHouse was built to apply my frontend development knowledge and improve my skills in creating real-world web applications with a focus on responsive design and user experience.
